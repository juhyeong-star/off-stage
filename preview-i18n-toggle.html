How It Works

우상단 KO / EN 버튼을 눌러보세요. 화면 안의 모든 정적 텍스트가 즉시 전환돼요. 새로고침 안 해도 적용되고, 새로고침 후에도 마지막 선택이 유지돼요.

Tap the KO / EN button at the top right. All static text on the page switches instantly. No refresh required, and your choice persists across reloads.

구현 방식 (참고)

Implementation (Reference)

방식: CSS 클래스 토글로 가장 가벼움. 두 언어 텍스트가 DOM 에 같이 있고 html[data-lang="..."] 룰이 한쪽을 숨김.

How: Lightweight CSS class toggle. Both KO & EN strings live in the DOM, hidden via html[data-lang="..."] rules.

장점: 깜빡임 없음 · 빠름 · 외부 라이브러리 X · 새 텍스트도 <span data-i18n-ko> / <span data-i18n-en> 두 개로 두면 끝.

Pros: No flicker · fast · zero dependencies · just add two spans <span data-i18n-ko> / <span data-i18n-en> for new strings.

단점: 현재 사이트에 한글 문자열이 약 2000여 곳 박혀있어서, 점진적으로 번역해야 함 (우선순위: 네비/인트로/버튼 → 폼/모달 → 알림).

Cons: ~2,000 Korean strings already embedded in the site — needs progressive translation (priority: nav/intro/buttons → forms/modals → toasts).
