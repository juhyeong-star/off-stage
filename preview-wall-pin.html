현재 (왼쪽) vs 3가지 제안 — 마음에 드는 것 골라주세요

현재 (BEFORE)

큰 노란 사각 + 그림자
특별한 장식 없음

오늘 새 곡 작업 중— 쿠츠
새벽 감성 좋다— 김
스튜디오 짱— 박
기분 좋은 하루— 이

A · 코르크보드 + 압정

갈색 보드 위 작은 노트 + 빨강/파랑 압정 + 살짝 기울어진 각도

오늘 새 곡 작업— 쿠츠
새벽 감성— 김
스튜디오 짱— 박
기분 좋은 하루— 이
곧 발매— 정
감사합니다— 최

B · 종이클립 묶음

노트 2장이 클립으로 묶임 — 답글/모음 느낌

오늘 새 곡 작업 — 쿠츠
새벽 감성 — 김
스튜디오 짱 — 박
기분 좋은 하루 — 이

C · 마스킹 테이프 띠

세로 1열, 작은 가로 노트 + 위에 마스킹 테이프 = 스크롤 잘됨

오늘 새 곡 작업 중— 쿠츠
새벽 감성 좋다 ✨— 김
스튜디오 짱— 박
기분 좋은 하루— 이
선택 가이드

A 코르크보드 + 압정: 가장 "벽" 느낌 살아남. 갈색 배경 + 작은 노트 + 압정.
B 종이클립: 답글/묶음 느낌. 2장이 한 세트로 보임.
C 마스킹 테이프: 스크롤 깔끔. 1열로 길게.

공통: 현재보다 노트가 작아짐 + 장식이 들어가서 덜 단조롭게 보임.
